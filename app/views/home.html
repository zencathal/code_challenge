The factorial of a number is computed as that number times all of the numbers below it up to and including 1. For example, factorial(5) is the same as 5*4*3*2*1, and factorial(3) is 3*2*1. Write a function that takes n as an argument and returns the factorial of n. You have five minutes to complete the task preferably in javascript or python.
